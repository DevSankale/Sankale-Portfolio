Project 2.

Calculator.

This calculator project consists of a visually appealing user interface designed using HTML and CSS.The CSS styling ensures the calculator looks neat and organized,JavaScript handles the functionality of the calculator. It captures user inputs, performs arithmetic calculations based on those inputs, and displays the result on the calculator's screen in real-time. Event listeners are used to detect button clicks and perform the corresponding actions, such as appending numbers or operators to the input, evaluating expressions and clearing the input.Overall, this simple calculator provides a user-friendly interface for basic arithmetic calculations, making it convenient and easy to use for simple math tasks.
